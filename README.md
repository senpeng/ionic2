    * 下载地址：[https://nodejs.org/en/](https://nodejs.org/en/)
    * 安装好后在命令行中输入 `node -v` 和 `npm -v` 查看是否安装成功

    C:\WINDOWS\system32>node -v
    v6.10.0

    C:\WINDOWS\system32>npm -v
    3.10.10`
